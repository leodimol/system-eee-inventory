Main Hub Functions
1
Receive Export Files
Main hub receives JSON/CSV export files from all operator hubs. Each file contains the hub's complete laptop inventory data.
File Format: { hubName, laptops: [], lastSync: timestamp }
2
Import & Validate
Upload files through Data Merge center. System validates file structure and shows preview of data to be merged.
Validation: Check required fields, unique IDs, data format
3
Merge Data
Execute merge to combine hub data into main database. System handles duplicate detection and conflict resolution.
Merge Logic: Append new records, update existing, flag conflicts
4
Update Aggregated Dashboard
All dashboards refresh to show combined data from all hubs. Stats, charts, and tables reflect merged inventory.
Dashboard Update: Total laptops, active count, maintenance status
5
Generate Reports
Create company-wide reports including inventory overview, hub comparison, maintenance analysis, and financial summaries.
Report Types: Overview, Hub Comparison, Maintenance, Financial
